API utilizando la librería axios y VUE

La API desarrollada nos devuelve los datos contenidos en un archivo propio de extensión .json alojado en nuestra terminal; y refieren a asentamientos humanos relevados de cartas topográficas, propiedad del IGN, en la zona de las Misiones Jesuíticas. Los datos nos devuelven el id, el nombre y el tipo de asentamiento.
